# 🔗 5~6회차 종합 비교: 두 논문 + 당신의 논문과의 관계

## 세 논문의 위치

```
[Robustness of Diff. CD]        [CausalCompass]
i.i.d. 환경에서                  시계열 환경에서
미분가능 CD의 강건성              TSCD 방법들의 강건성
이론 설명 포함                    전처리 효과 포함
        │                               │
        └───────── 공통 공백 ────────────┘
                        │
           "비선형 관측 변환 후 차분이
            인과 탐색 성능에 미치는 효과"
                        │
               [당신의 논문 포지셔닝]
```

## 당신 논문이 두 논문과 다른 것

| 항목 | CausalCompass | Robustness of Diff. CD | **당신의 논문** |
|---|---|---|---|
| 위반 유형 | 메커니즘 위반 | 메커니즘 위반 | **관측 변환 위반** |
| 전처리 종류 | 표준화/정규화 | 스케일 변형 | **차분(differencing)** |
| 데이터 타입 | 시계열 | i.i.d. | 시계열 |
| 이론 포함 | ❌ | ✅ | ❌ (현재) |
| 변환 함수 분석 | ❌ | ❌ | ✅ (tanh/포화 함수) |

## 당신 논문이 지금 당장 채울 수 있는 공백

> **"비선형 관측 변환 X_t = f(Z_t) 환경에서, 차분이라는 전처리가 NOTEARS 기반 인과 탐색의 SHD를 개선하는가? 그리고 이 효과는 f의 수학적 성질(단조성, 포화도)에 어떻게 의존하는가?"**

이 질문은 두 논문 모두 명시적으로 다루지 않습니다. CausalCompass는 차분을 시나리오에 포함하지 않았고, Robustness of Diff. CD는 관측 변환을 시나리오에 포함하지 않았습니다. 당신의 논문이 정확히 이 틈새에 들어갑니다. 단, 앞에서 지적한 대로 **이론 설명 한 줄**이 없으면 이 두 논문보다 약한 논문이 됩니다.
