# 🔗 CD-NOTS vs CaRiNG — 두 논문의 관계 & 연구 연결고리

## 1. 두 논문이 공유하는 큰 그림

두 논문은 같은 문제 공간에서 출발하지만, **서로 다른 층위**에서 접근합니다.

```
[실제 세계]
    ↓ 생성 과정 (mixing / observation)
[관측 데이터 x_t]  ← CD-NOTS가 작동하는 층
    ↓ 표현 학습 (representation learning)
[잠재 변수 z_t]   ← CaRiNG이 작동하는 층
    ↓ 인과 구조 탐색
[인과 그래프 G]    ← 두 논문 모두 궁극적 목표
```

즉, **CD-NOTS는 관측 공간에서 직접 인과 구조를 찾고**, **CaRiNG은 관측에서 잠재 변수를 먼저 복원한 뒤** 그 구조를 이해하려 합니다.

---

## 2. 핵심 차원별 비교

| 비교 축 | CD-NOTS | CaRiNG |
|---|---|---|
| **분야** | 인과 탐색 (Causal Discovery) | 인과 표현 학습 (Causal Representation Learning) |
| **입력** | 관측 시계열 `x_t` (표 형태) | 시계열 `x_t` (이미지/고차원 가능) |
| **출력** | 인과 그래프 (DAG) | 잠재 인과 변수 `z_t` + 인과 구조 |
| **핵심 도전** | 비정상성 (분포가 시간 따라 변함) | 비가역성 (정보 손실 있는 생성 과정) |
| **핵심 도구** | 조건부 독립 검정 + 시간 인덱스 노드 | 비선형 ICA + Sequential VAE + 시간 맥락 |
| **가정** | 관측 변수 = 인과 변수 | 관측 변수 ≠ 인과 변수 (잠재 변수 존재) |
| **데이터 타입** | 표 형태 시계열 (금융, 경제, 의료) | 고차원 시계열 (영상, 복잡한 센서) |
| **이론적 보장** | 점근적 그래프 복원 | 잠재 변수 식별가능성 (up to permutation) |
| **계산 방식** | 비모수적 통계 검정 | 딥러닝 (VAE 기반) |

---

## 3. 두 논문이 서로 보완하는 지점

### CD-NOTS의 한계 → CaRiNG이 채운다
CD-NOTS는 **관측 변수 자체가 인과 변수라고 가정**합니다. 하지만 실제 데이터(특히 영상, 센서)에서는 관측값 뒤에 숨겨진 잠재 변수가 진짜 인과 주체인 경우가 많습니다. CaRiNG은 이 잠재 변수를 먼저 복원해서 더 근본적인 인과 구조를 찾을 수 있습니다.

### CaRiNG의 한계 → CD-NOTS가 채운다
CaRiNG은 잠재 변수를 복원하지만, **변수 간 구체적인 인과 방향과 지연 구조(어떤 변수가 몇 타임스텝 뒤 무엇에 영향을 주는가)를 명시적으로 표현하고 검정하는 능력은 상대적으로 약합니다.** CD-NOTS의 조건부 독립 검정 프레임워크는 이 부분을 통계적으로 엄밀하게 다룹니다.

---

## 4. 석사 논문 연구에서의 활용 방향

두 논문을 읽고 있다는 맥락에서, 가능한 연구 포지셔닝을 몇 가지로 그려볼 수 있습니다.

**방향 A — CaRiNG 확장: 동시 인과관계(instantaneous relation) 처리**
CaRiNG의 가장 명확한 한계는 동시 인과관계를 처리하지 못한다는 점입니다. CD-NOTS가 동시 효과와 지연 효과를 모두 탐색하는 것처럼, CaRiNG의 잠재 공간에서도 이를 처리하는 것이 자연스러운 확장 방향입니다.

**방향 B — 두 방법의 파이프라인 연결**
CaRiNG으로 잠재 변수를 복원한 뒤, 그 잠재 시계열에 CD-NOTS를 적용해 인과 그래프를 탐색하는 2단계 파이프라인을 설계할 수 있습니다. 이 경우 CaRiNG이 고차원→저차원 압축을, CD-NOTS가 구조 탐색을 담당하는 역할 분담이 이루어집니다.

**방향 C — 비정상성 + 비가역성 동시 처리**
CD-NOTS는 비정상성을, CaRiNG은 비가역성을 각각 처리하지만, **두 문제가 동시에 존재하는 실제 데이터**에 대응하는 통합 프레임워크는 아직 연구 공백입니다.

**방향 D — 도메인 특화 적용 및 검증**
금융 데이터처럼 비정상성이 강한 환경에서 CD-NOTS를 기준선으로, 영상이나 복잡한 센서 데이터처럼 비가역성이 강한 환경에서 CaRiNG을 기준선으로 삼아, 특정 도메인에서 두 방법의 성능 경계를 실험적으로 검증하는 방향도 가능합니다.

---

## 5. 핵심 요약 한 문장

> **CD-NOTS는 "관측된 시계열에서 통계적으로 인과 구조를 찾는 방법"이고, CaRiNG은 "고차원 관측 뒤에 숨겨진 잠재 인과 변수를 복원하는 방법"입니다. 두 논문은 같은 목표(시계열 인과 이해)를 서로 다른 층위에서 공략하며, 함께 사용하면 더 완결된 파이프라인을 구성할 수 있습니다.**

## 4. 세 논문이 함께 열어주는 연구 공간

세 논문을 동시에 놓고 보면, 아직 채워지지 않은 공백이 선명하게 드러납니다.

**공백 A — "명시적 차분 → 인과 구조" 연결고리의 부재**
DyMoDreamer는 차분을 가장 명시적으로 사용하지만, 그 차분이 어떤 인과 관계를 드러내는지는 탐구하지 않습니다. CD-NOTS의 인과 탐색 프레임워크를 DyMoDreamer의 동적 조절 변수 위에 얹으면, **"어떤 객체의 변화가 어떤 보상에 인과적으로 연결되는가"** 를 해석할 수 있게 됩니다.

**공백 B — 비가역 환경에서의 차분 기반 인과 탐색**
CaRiNG은 비가역적 생성(3D→2D)에서 잠재 변수를 복원하지만, 그 잠재 공간에서 DyMoDreamer식 차분을 적용하면 더 안정적인 동적 특성 추출이 가능할 수 있습니다.

**공백 C — 비정상 환경에서의 동적 조절**
DyMoDreamer는 환경의 동역학이 시간에 따라 변하는(비정상) 경우를 명시적으로 다루지 않습니다. CD-NOTS의 비정상성 처리와 DyMoDreamer의 차분 마스크를 결합하면, **시간에 따라 중요한 동적 객체가 바뀌는 환경**에 대응할 수 있습니다.

*"DVAE: A Dynamic Variational Autoencoder for Structured Causal Discovery with Application in Biomedical Time Series"*
DOI: 10.1016/j.knosys.2025.115154

---

## 1. 기본 요약

### 논문이 풀려는 문제

시계열 데이터에서의 인과 탐색은 신경과학, 경제학, 생의학 신호 처리에서 동적 시스템을 분석하는 데 핵심적입니다. VAR 같은 전통 방법과 제약 기반 접근법은 고차원 의존성, 비선형 관계, 비정상 동역학에서 어려움을 겪습니다. 딥러닝 기반 모델들(cMLP, cLSTM, VAE 기반 접근법)도 불안정성, 과다 가지치기(over-pruning), 희소성 제약에 대한 의존이라는 문제를 안고 있으며, cMLP는 시차별(lag-specific) 인과 추론을 제공하지만 정확도가 제한적이고 다른 방법들은 시차별 의존성을 명시적으로 포착하지 못합니다.

### DVAE-GC란?

DVAE-GC는 동적 변분 추론(dynamic variational inference)과 시차 구조화된 순환 MLP(lag-structured recurrent MLP, lsrMLP)를 통합한 구조화된 딥러닝 프레임워크로, 시간 지연 인과 의존성을 명시적으로 모델링합니다. 가중치 희소성으로 인과성을 추론하는 기존 방법들과 달리 인과 추정을 점진적으로 정제(progressively refine)하며, 양방향 순환 인코더와 구조화된 디코더를 활용합니다. 추가로 Noise Invalidation Soft Thresholding(NIST)이 허위 연결을 제거하여 해석가능성과 강건성을 높입니다.

---

## 2. 핵심 방법론: 4가지 구성 요소

DVAE-GC는 VAE 프레임워크 위에 Granger 인과성을 명시적으로 내재화한 구조입니다.

### 핵심 아이디어: Granger 인과성과 VAE의 결합

**Granger 인과성의 정의:** 변수 X의 과거값이 Y의 예측에 통계적으로 유의미하게 기여한다면, X는 Y에 Granger 인과적입니다. 즉, `Y_t`를 예측할 때 `X_{t-1}, X_{t-2}, ..., X_{t-k}`를 포함하면 포함하지 않은 것보다 예측 오차가 줄어드는지를 검정합니다.

DVAE-GC는 이 아이디어를 VAE의 디코더 구조 안에 직접 내장합니다.

### Step 1 — 시차 구조화된 순환 MLP (lsrMLP) 디코더

```
시계열 x_t를 예측할 때:
각 변수 x_t^p는 다른 모든 변수들의 특정 시차(lag k)값에만
선택적으로 의존하도록 디코더 구조를 설계

x_t^p = f(x_{t-1}^1, x_{t-2}^1, ..., x_{t-K}^N, z_t)
         ↑
  어떤 연결이 활성화되는지가 = 인과 그래프
```

이전 VAE 기반 방법들이 디코더 가중치 희소성으로 간접적으로 인과성을 추론했다면, DVAE-GC는 lsrMLP를 통해 **시차별 인과 연결을 구조 자체에 명시화**합니다.

### Step 2 — 양방향 순환 인코더 (Bidirectional Recurrent Encoder)

과거뿐 아니라 미래 맥락도 활용하여 잠재 변수 `z_t`를 더 정확하게 추정합니다. 양방향 RNN이 시계열을 앞뒤로 스캔함으로써 단방향 인코더보다 더 풍부한 시간적 문맥을 포착합니다.

### Step 3 — 점진적 인과 추정 (Progressive Causal Refinement)

기존 방법들이 단일 패스로 인과 그래프를 추정하는 것과 달리, DVAE-GC는 **학습 과정에서 인과 연결을 점진적으로 정제**합니다. 초기에는 많은 연결을 허용하고, 학습이 진행될수록 약한 연결을 점차 제거하는 방식으로 over-pruning 문제를 완화합니다.

### Step 4 — NIST (Noise Invalidation Soft Thresholding)

추정된 인과 그래프에서 허위 연결(spurious connections)을 제거하여 해석가능성과 강건성을 높이는 후처리 단계입니다. 단순한 하드 임계값 절단 대신 소프트 임계값을 적용해 노이즈에 의한 약한 연결을 부드럽게 억제합니다. 이는 저자들의 이전 연구(2023년 NIST 논문)에서 발전시킨 기법입니다.

### 전체 구조 요약

```
다변량 시계열 {x_1, ..., x_T}
        ↓
[양방향 순환 인코더]
  → 잠재 변수 z_t 추정 (앞+뒤 맥락 모두 활용)
        ↓
[lsrMLP 디코더]
  → 각 변수를 특정 시차의 다른 변수들로 예측
  → 어떤 시차 연결이 활성화되는지 = 인과 구조
        ↓
[ELBO 최적화 + 점진적 정제]
        ↓
[NIST 후처리]
  → 허위 연결 제거 → 최종 인과 그래프
```

| 잠재 변수 활용 | ❌ | ❌ | ✅ | ✅ |
| 허위 연결 제거 | 임계값 | 임계값 | 희소성 | ✅ NIST |
| 양방향 시간 인코딩 | ❌ | ❌ | ❌ | ✅ |
| 도메인 | 다양 | 다양 | 의료 | **생의학 특화 + 범용** |

### 실험 결과

VAR(9) 벤치마크에서 최고 기준선(CUTS) 대비 F1-score 18.3% 향상(노이즈 수준 전반 평균), NetSim fMRI-20 데이터셋에서 8.1% F1-score 향상, 시뮬레이션된 심방 로터(atrial rotor) 탐지에서 최고 대안(VAR) 대비 Rotational Activity Estimation Precision(RAEP) 22.4% 향상을 달성했습니다. 또한 precision, recall, false discovery rate(FDR)도 함께 개선되었습니다.

## 4. 한계

**① 비정상성 명시 처리 부재**
전통 방법들이 비정상 동역학에서 어려움을 겪는다고 지적하면서도, DVAE-GC 자체가 비정상성을 명시적으로 모델링하는 메커니즘(CD-NOTS의 시간 인덱스 노드 같은 것)을 갖추고 있지 않습니다. 시간에 따라 인과 구조 자체가 변하는 경우 대응이 제한적입니다.

**② 생의학 시뮬레이션 중심 검증**
생의학 시뮬레이션에서 평가되었지만, DVAE-GC는 신경과학, 기후 과학, 금융 모델링을 포함한 광범위한 시계열 도메인에 적용 가능합니다. 그러나 실제 금융·기후 데이터에서의 검증은 아직 이루어지지 않았습니다.

**③ Granger 인과성의 근본적 한계 계승**
Granger 인과성은 예측력(predictive power) 기반이므로, 잠재 공통 원인(confounders)이 있거나 측정되지 않은 변수가 있는 경우 인과 방향을 잘못 추론할 수 있습니다. 이는 DVAE-GC가 상속하는 기본 한계입니다.

**④ 계산 비용**
양방향 순환 인코더 + lsrMLP 디코더 + 점진적 정제를 함께 사용하므로 단순 VAR이나 cMLP 대비 계산 비용이 높습니다.

## 3. "차분" 프레임으로 본 네 논문 업데이트

| 논문 | 차분 사용 방식 | 차분이 드러내는 것 |
|---|---|---|
| **CD-NOTS** | 시간 인덱스 T = 연속 차분의 부드러운 추상화 | 분포 변화 신호 → 인과 방향 |
| **CaRiNG** | 과거 맥락 = 다단계 차분 패턴의 암묵적 학습 | 손실된 잠재 차원 복원 |
| **DyMoDreamer** | 픽셀 차분 마스크 = 동적 객체의 명시적 분리 | 보상 관련 동역학 신호 |
| **DVAE-GC** | lsrMLP의 시차 구조 = lag-k 차분의 구조화된 모델링 | Granger 인과 그래프 |

DVAE-GC가 네 번째 논문으로 가져오는 가장 중요한 새 관점은 **"차분을 특정 시차(lag k)로 구조화하여 인과 그래프의 엣지 자체로 직접 매핑"** 한다는 점입니다. 이는 당신의 연구 테제 — *차분이 latent causal structure를 드러내는 연산* — 의 가장 직접적인 구현체에 해당합니다.

## 4. 네 논문이 함께 가리키는 연구 공백

네 논문을 모두 놓고 보면 다음 공백이 선명해집니다:

> **"비정상 + 비가역 + 고차원" 세 조건이 동시에 성립하는 시계열에서, 차분 기반 lag 구조를 통해 잠재 인과 그래프를 식별하는 통합 프레임워크"**

- CD-NOTS: 비정상성 ✅, 비가역성 ❌, 고차원 ❌
- CaRiNG: 비정상성 ❌, 비가역성 ✅, 고차원 ✅
- DyMoDreamer: 비정상성 ❌, 비가역성 △, 고차원 ✅
- DVAE-GC: 비정상성 ❌, 비가역성 ❌, lag 구조 ✅

세 조건을 동시에 다루면서 lag 구조의 차분을 인과 신호로 활용하는 방법은 아직 공백입니다.
