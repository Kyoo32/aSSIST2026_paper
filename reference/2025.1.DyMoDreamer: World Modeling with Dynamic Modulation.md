## 1. 기본 요약

### 논문이 풀려는 문제

딥 강화학습(DRL)의 핵심 병목은 샘플 비효율성입니다. 고성능 에이전트를 훈련하려면 환경과의 방대한 상호작용이 필요합니다. 모델 기반 강화학습(MBRL)은 환경 동역학을 시뮬레이션하는 월드 모델을 구축하고 합성 경험을 생성함으로써 이 문제를 완화합니다.

그러나 기존 월드 모델은 관측을 전체적(holistically)으로 처리하여, 동적 객체와 시간적 특성을 정적 배경으로부터 분리하는 데 실패합니다. 이는 특히 동적 객체가 보상과 의사결정에 크게 영향을 미치는 시각적 태스크에서 계산적으로 비효율적입니다.

### DyMoDreamer란?

인간 영아가 주변의 기본 원리를 추론하기 위해 동적 객체 상호작용에 자연스럽게 집중하고, 보상은 주로 동적 부분에 의해 영향을 받는다는 인지과학적 통찰에서 출발합니다. 이를 바탕으로, 동적 조절(dynamic modulation) 메커니즘을 활용하여 월드 모델링 과정을 강화하는 새로운 MBRL 알고리즘입니다. Atari 100k에서 156.6% mean human-normalized score, DeepMind Visual Control Suite에서 832점, Crafter에서 1M 스텝 기준 9.5% 성능 향상으로 새로운 SOTA를 달성했습니다.

## 2. 핵심 방법론: 3가지 구성 요소

DyMoDreamer는 DreamerV3를 기반으로 세 가지 핵심 요소를 추가합니다.

### Step 1 — 프레임 간 차분 마스크 (Inter-Frame Differencing Mask)

차분 관측(differential observations)을 새로운 프레임 간 차분 마스크로부터 도출하여, 객체 수준의 모션 단서와 시간적 동역학을 명시적으로 인코딩합니다.

핵심 직관은 단순합니다. `Δo_t = o_t − o_{t-1}`에서 0에 가까운 픽셀은 정적 배경이고, 큰 변화가 있는 픽셀은 움직이는 동적 객체입니다. 이 차분값을 마스크로 변환하여 월드 모델이 **어디를 집중해야 하는지**를 명시적으로 알려줍니다.

### Step 2 — RSSM에 동적 조절 변수(Dynamic Modulator) 통합

월드 모델은 RSSM 위에 구축되며, 시퀀스 모델은 순환 잠재 상태 `h_t`를 활용해 과거 행동 `a_{t-1}`에 기반한 확률적 표현과 동적 조절 변수 시퀀스를 예측합니다. 모델 상태는 `h_t`, `z_t`, `d_t`를 연결(concatenate)하여 형성되며, 이를 통해 보상 `r_t`와 에피소드 종료 여부를 예측합니다.

즉 기존 RSSM의 상태가 `(h_t, z_t)`였다면, DyMoDreamer는 여기에 동적 조절 변수 `d_t`를 추가하여 **"지금 환경에서 무엇이 움직이고 있는가"** 를 상태의 일부로 명시화합니다.

### Step 3 — 차분 발산 정규화 (Differential Divergence Regularization, L_reg)

차분 발산 정규화 `L_reg`는 주로 프레임 내(intra-frame) 차이에 집중하는 MSE 손실의 한계를 보완합니다. 후방 차분(backward difference) `Δô_t = ô_t − ô_{t-1}`과 `Δo_t = o_t − o_{t-1}`을 계산하고, 이를 소프트맥스를 통해 확률 분포로 변환한 뒤 두 분포 간의 KL 발산을 정규화 항으로 사용합니다. 이를 통해 프레임 간(inter-frame) 변화에 대한 인식을 높입니다.

### 전체 구조 요약

```
관측 o_t
    ↓
[Inter-Frame Differencing Mask]
    → Δo_t = o_t − o_{t-1} → 동적 영역 마스크 추출
    ↓
[RSSM + Dynamic Modulator d_t]
    모델 상태: (h_t, z_t, d_t)
    → 보상, 동역학, 다음 상태 예측
    ↓
[학습 손실]
    = Prediction Loss + Dynamic & Representation Loss
    + Differential Divergence Regularization (L_reg)
    ↓
[Latent Imagination으로 정책 학습]
```

모든 구성 요소는 별도의 조절 변수 훈련 없이 함께 end-to-end로 훈련됩니다.

## 3. 기존 연구(DreamerV3)와의 차이점

| 항목 | DreamerV3 | DyMoDreamer |
|---|---|---|
| 관측 처리 방식 | 전체 프레임을 동등하게 처리 | 차분 마스크로 동적/정적 영역 분리 |
| 잠재 상태 | `(h_t, z_t)` | `(h_t, z_t, d_t)` — 동적 조절 추가 |
| 손실 함수 | MSE + KL | MSE + KL + `L_reg` (차분 KL 발산) |
| 환각(hallucination) | 동적 패턴에서 환각 발생 | 동적 패턴 환각 대폭 감소 |
| 성능 (Atari 100k) | 기준선 | 156.6% mean human-normalized score |

특히 보상과 관련된 핵심 객체가 희소하고 서로 독립적인 환경(예: Pong, Finger Spin)과, 태스크 단계 간 전환이 뚜렷한 환경(예: Krull, Crafter)에서 동적 조절 변수의 정확한 포착이 강점을 발휘합니다.

### Ablation 핵심 발견

동적 조절 변수의 차원만 단순히 늘리는 것은 제한적인 이득만 줍니다. 동적 조절이 효과를 발휘하려면 시퀀스 모델(RSSM)에 내재화되어야 합니다. 또한 차분 발산 정규화를 제거하면 성능이 저하되는데, 이는 `L_reg`가 재구성 목표와 상호 보완적인 감독 신호임을 보여줍니다.

## 4. 한계

**① 차분 마스크의 단순성**
카메라가 에이전트와 함께 이동하는 환경에서는 차분 관측이 에이전트와 환경 구성 요소 간의 상대적 움직임을 포착하게 됩니다. 즉 카메라 자체의 이동(egomotion)이 큰 환경에서는 차분값이 동적 객체를 넘어 배경까지 포함해 노이즈가 될 수 있습니다.

**② 정적인 환경에서의 이점 불명확**
보상이 동적 정보로 완전히 포착되지 않는 환경(예: Gopher)에서도 일부 성능 향상이 있으나, 정적 특성이 핵심인 태스크에서 차분 기반 접근의 이점은 제한적입니다.

**③ 인과 구조에 대한 해석 부재**
동적 조절 변수 `d_t`가 어떤 객체, 어떤 인과 관계를 포착하고 있는지에 대한 해석가능성(interpretability)이 부족합니다. 차분이 동적 신호를 드러내지만, 그 신호가 인과적으로 무엇을 의미하는지는 모델이 암묵적으로 학습합니다.

**④ 시각적 관측 특화**
이 방법은 픽셀 수준의 프레임 차분을 전제로 하므로, 표 형태 데이터나 벡터 관측 환경에는 직접 적용이 어렵습니다.
