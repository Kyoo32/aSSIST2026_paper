# 📄 CaRiNG 논문 분석 — 2회차

**Chen et al., arXiv 2401.14535, ICML 2024**
*"CaRiNG: Learning Temporal Causal Representation under Non-Invertible Generation Process"*
저자: Guangyi Chen, Yifan Shen, Zhenhao Chen, Xiangchen Song, Yuewen Sun, Weiran Yao, Xiao Liu, Kun Zhang

---

## 1. 기본 요약

### 논문이 풀려는 문제

시계열 데이터에서 시간 지연(time-delayed) 잠재 인과 과정을 식별하는 것은 시간적 동역학을 파악하고 다운스트림 추론을 수행하는 데 핵심적입니다. 기존의 방법들은 잠재 변수에서 관측 데이터로의 **가역적(invertible) 생성 과정**을 가정하는데, 이 가정은 정보 손실이 발생하는 실제 문제에서 충족되기 어렵습니다.

예를 들어, 3D 공간이 2D 이미지로 투영되는 시각 인식 과정이나, 잔상 현상(persistence of vision)처럼 과거 정보가 현재 인식에 혼합되는 경우가 대표적인 비가역적(non-invertible) 생성 과정입니다.

### CaRiNG이란?

**Ca**usal **R**epresentat**i**on of **N**on-invertible **G**enerative temporal data의 약자로, 비가역적 생성 과정 하에서도 식별가능성 보장(identifiability guarantees)과 함께 잠재 인과 표현을 학습하는 원칙적 접근법입니다. 시간적 맥락(temporal context)을 활용해 손실된 잠재 정보를 복원하는 것이 핵심 아이디어입니다.

---

## 2. 핵심 이론 및 방법론

### 왜 기존 방법이 실패하는가? — 가역성 가정의 문제

기존의 비선형 ICA 기반 시간적 인과 표현 학습(TDRL, LEAP 등)은 잠재 변수 `z_t` → 관측값 `x_t`로의 혼합 함수(mixing function) `f`가 **가역적**이라고 가정합니다. 즉, `x_t = f(z_t)`일 때 `f`의 역함수가 존재해야 `z_t`를 복원할 수 있습니다.

비가역적 환경에서 강한 기준선인 TDRL은 MCC가 해당 차원에서 0.03까지 떨어지며, 비가역적 생성으로 인해 손실된 잠재 변수를 복원하지 못합니다. 반면 시간적 맥락을 사용하는 Sequential VAE 방법이 상대적으로 더 잘 동작한다는 점도 관찰되는데, 이는 역가역성 문제를 해결하기 위해 시간적 맥락의 필요성을 방증합니다.

### 핵심 이론적 기여: 비가역적 생성 하의 식별가능성 정리

이 논문은 비선형적이고 비가역적인 혼합으로부터 나온 경우에도 독립적인 잠재 성분들을 복원할 수 있음을 증명한 **최초의 식별가능성 정리**를 제시합니다.

핵심 직관은 다음과 같습니다. 현재 시점의 관측값 `x_t`만으로는 비가역적 혼합 때문에 `z_t`를 완전히 복원할 수 없지만, **과거 시점들의 맥락 `{x_{t-1}, x_{t-2}, ...}`을 함께 활용하면** 손실된 잠재 정보를 보완적으로 복원할 수 있습니다.

### 알고리즘 구조: Sequential VAE + 두 가지 핵심 수정

CaRiNG은 Sequential VAE를 기반으로 두 가지 핵심적인 수정을 가합니다. 첫째, 역사적(historical) 맥락 정보를 인코더에 직접 통합합니다. 둘째, 이론의 조건들을 학습 과정을 안내하는 정규화(regularization)로 적용합니다. 구조를 단계별로 정리하면:

| 단계 | 내용 |
|---|---|
| **인코더** | 현재 관측 `x_t` + 과거 맥락 `{x_{t-τ}, ..., x_{t-1}}`을 함께 입력받아 잠재 변수 `z_t` 추정 |
| **전이 모델** | 시간 지연 인과 구조 `z_{t-k} → z_t`를 모델링 |
| **디코더** | `z_t`로부터 `x_t` 재구성 |
| **학습 목표** | ELBO + 식별가능성 조건에서 유도된 정규화 항 |

## 3. 기존 연구와의 비교

| 항목 | 기존 방법 (TDRL, LEAP 등) | CaRiNG |
|---|---|---|
| 혼합 함수 가정 | 가역적(invertible) ✅ 필요 | 비가역적(non-invertible) ✅ 허용 |
| 시간적 맥락 활용 | ❌ 현재 시점만 사용 | ✅ 과거 맥락으로 손실 정보 복원 |
| 식별가능성 보장 | 가역적 환경에서만 | 비가역적 환경에서도 이론적 보장 |
| 기반 이론 | 비선형 ICA | 비선형 ICA + 새로운 비가역성 정리 |
| 동시 인과관계 | 대부분 제외 | 제외 (시간 지연만 처리) |

CaRiNG은 비선형 ICA 기반의 이론적 결과에 의해 동기 부여되어, 맥락 관측값을 활용해 손실된 잠재 정보를 복원하고 비가역적 혼합 함수의 도전을 해결합니다.

### 실험 결과

비가역적 생성(NG) 환경에서 CaRiNG은 MCC 0.933 ± 0.010을 기록하며, TDRL(0.627), LEAP(0.651), SlowVAE(0.362) 등 기존 방법들을 큰 폭으로 능가했습니다.

## 4. 한계

**① 동시적(instantaneous) 인과관계 미지원**
대부분의 시간적 인과 표현 학습 방법들과 마찬가지로, CaRiNG도 잠재 인과 과정이 동시 관계(instantaneous relations)를 갖지 않는다는 가정에 의존합니다. 즉, 같은 시간 단계 내에서 발생하는 즉각적 인과 효과를 처리하지 못합니다.

**② 합성 데이터 중심의 실험**
실험은 주로 합성 데이터셋에서의 MCC 측정으로 평가되며, 실세계 영상 QA 과제(SUTD-TrafficQA)에서 추가 검증이 이루어졌지만 다양한 실제 도메인(기후, 금융, 의료 등)에서의 광범위한 검증은 제한적입니다.

**③ 계산 비용**
과거 맥락 전체를 인코더에 입력하는 방식은 시퀀스가 길어질수록 메모리와 연산 비용이 선형 이상으로 증가할 수 있습니다.

**④ 혼합 함수의 부분적 가정**
비가역적 혼합은 허용하지만, 여전히 혼합 함수가 특정 수학적 조건(정리의 조건들)을 만족해야 합니다. 완전히 임의적인 비가역 혼합에 대한 보장은 없습니다.

## 핵심 용어 정리

**Sequential VAE (순차적 변분 오토인코더)**
VAE(Variational Autoencoder)는 데이터를 잠재 공간(latent space)으로 압축하는 인코더와, 잠재 변수에서 데이터를 재구성하는 디코더로 이루어진 생성 모델입니다. Sequential VAE는 이를 시계열로 확장한 것으로, 각 시간 단계의 잠재 변수 `z_t`가 이전 시점들의 정보에 의존하도록 설계됩니다. 쉽게 말하면 "현재 상태를 이해하기 위해 과거를 기억하는 VAE"입니다. CaRiNG은 이 Sequential VAE를 기반으로, 인코더에 과거 관측값들의 맥락을 추가로 주입하고 이론 기반 정규화를 더해 잠재 변수의 인과적 식별가능성을 보장하도록 확장합니다.

**비가역적(Non-invertible) vs 가역적(Invertible) — 그리고 "역가역성"에 대해**
"역가역성"은 논문 요약에서 비가역성(non-invertibility)을 지칭하는 표현으로, 함수의 역함수가 존재하지 않는 성질을 말합니다. 가역적(invertible) 혼합 함수 `f`는 `x = f(z)`가 주어졌을 때 `z = f⁻¹(x)`로 잠재 변수를 완전히 복원할 수 있지만, 비가역적(non-invertible) 혼합 함수는 역함수가 존재하지 않아 `x`만 가지고는 `z`를 복원할 수 없습니다. 3D 공간을 2D 이미지로 투영하는 것이 대표적인 예인데, 사진 한 장만 보고는 피사체의 깊이(depth)를 정확히 알 수 없는 것과 같은 이치입니다. CaRiNG의 핵심 기여는 바로 이 비가역적 상황에서도, 여러 시점의 관측값을 함께 활용하면 손실된 잠재 정보를 복원할 수 있음을 이론적으로 보장한 것입니다.
